# European Tech Internships 2024
#### A collection of tech internships around europe.

Results are a mix of scraping from linkedin and github and users contributions.

## Scraper
Scraper repository will be released soon.

## Contributions
You can contribute in two ways:
- adding a line with a linkedin link of a relevant job post in ```contribute/linkedin.txt```
- adding a line with a indeed link of a relevant job post in ```contribute/indeed.txt```
- adding a line with informations in the format ```<company>, <job post title>, <place>, <link>``` in ```contribute/lines.txt```

Note: do not care if your link already exists in either the case, no duplicate lines will be output
